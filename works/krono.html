Krono

Description

My team of five OIT students created Krono as our nine-month Junior project. It was developed for the Android platform, and it was created to help people create plans together easier.

Krono was designed with a strong focus on social features. Our ambitious scope included things like public events, an interactive realtime map with friend locations visible (think Snapchat's Snapmap feature), collaboration requests, and an intuitive calendar view to display plans.

With the time we had, we were able to produce a stable and intuitive planning app with friend features and collaboration. Activities also have geolocation attributes, allowing them to be easily located using the integrated Google Maps features.

Technologies

  • Java Programming Language

  • Android Studio

  • Google Firebase

  • Cloud Firestore

  • Firebase Authentication

  • Google Maps API
