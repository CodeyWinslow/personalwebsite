Description

I was graced with a remote summer internship by the Oregon NASA Space Grant Consortium to work with virtual reality technologies July 2020.

During the 10-week mentored project, I developed a touchscreen interface to be used in virtual reality in order to observe how well VR technologies promote rapid learning.

Because this project requires human subjects to gather data on virtual reality, IRB approval was required. This approval was not acquired before the internship deadline, but there are still plans to perform trials once I have approval.
